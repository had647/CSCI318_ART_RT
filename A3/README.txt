To set up GREP testing tool:
cd to A3 directory via Terminal in Ubuntu
compile java files by typing:  'javac Main.java GUIWindow.java ART.java' 
run Main.class in Terminal by typing:	'java Main'
Once user interface appears, change 'Path to Older GREP executable' and 'Path to GREP Oracle executable' to the current directory by clicking 'Open File' buttons accordingly.
